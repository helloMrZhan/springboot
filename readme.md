## 项目说明
本项目是一个用来spring boot学习和实践的项目，虽然spring boot3.0+版本已发布，
但是对于[JDK版本要求提升到了JDK 17+](https://spring.io/blog/2022/01/20/spring-boot-3-0-0-m1-is-now-available)，考虑到大多数人还是使用的JDK8,所以本项目采用的spring boot版本为2.7.11
